# biovoice
タンパク質のアミノ酸配列を読み取りそれに合わせた音を出します、【それぞれのタンパク質の音を聞いてみよう】というアプリです。
実行環境
biopython	1.78
numpy	1.19.3

実行時にはパスを通す。あらかじめfastaファイルをbiovoice.pyと同じディレクトリに入れる。
$ python biovoice.py (読みたいタンパク質).fastaで実行
